HTML is a markup language and also forms the backbone of all sites in the world.
HTML has no appearance and is styled by CSS.

The basic structure of an HTML document is as follows : <!DOCTYPE html><html><head><title></title></head><body> </body> </html>

Tags are the beginning and end parts of an element in HTML. And they start with <> . Elements store content between tags. And they generally consist of a start and end tag and the content that is between them.

Technically, an HTML element is the collection of start tag, its attributes, an end tag and everything in between. On the other hand an HTML tag (either opening or closing) is used to mark the start or end of an element.

href, link, width, height, color, background, font-size...

We use them inside the tags or put in links or files.
